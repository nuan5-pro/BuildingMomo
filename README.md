## 🎯 项目简介

BuildingMomo（搬砖吧大喵）

▸ 快速移动/复制/删除大型建筑群

▸ 在不同家园方案间自由合并建筑群

▸ 可视化编辑坐标，所见即所得

专为有特殊建造需求的玩家设计（如复杂建筑群调整、多方案整合、快速成组）

## 🚀 快速开始

### 🌐 在线使用

- [Cloudflare Pages](https://infinitymomo.com)
- [Github Pages](https://chaniok.github.io/BuildingMomo)

### 💻 本地运行

如果您是开发者或希望本地部署：

```bash
# 安装依赖
npm install

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

Vue 3 · Vite · Three.js · Tresjs · Pinia · Tailwind CSS · shadcn-vue
